# 🧠 抽象人格测试产品 PRD（V1.0）

## 一、产品概述

### 1.1 产品名称

**你是什么抽象人格（ABTI测试）**

### 1.2 产品定位

一个基于互联网文化的轻量级人格测试产品，通过“抽象梗人格 + 类MBTI编码”，让用户获得强共鸣、强分享欲的测试结果。

### 1.3 核心目标

* 提供有趣、共鸣感强的人格测试体验
* 激发用户主动分享（朋友圈 / 小红书 / 抖音）
* 实现低成本裂变传播

---

## 二、核心玩法

### 2.1 测试流程

```text
进入首页
↓
点击开始测试
↓
完成20道选择题
↓
计算人格维度分数
↓
生成ABTI人格编码（4位字母）
↓
匹配人格类型（13种）
↓
展示结果页（支持分享）
```

---

### 2.2 人格系统（ABTI）

#### 维度设计（4维）

| 维度 | 左     | 右     |
| -- | ----- | ----- |
| 情绪 | C（冷静） | X（爆炸） |
| 社交 | S（潜水） | A（活跃） |
| 思维 | L（理性） | B（抽象） |
| 表达 | H（自嘲） | P（自信） |

#### 编码示例

* XABP → 嘉豪型抽象圣体
* CSLP → 冷脸审判官型

---

### 4.1 页面结构

## 五、裂变玩法设计

### 5.1 基础分享裂变

用户生成结果 → 分享 → 新用户进入测试

---

### 5.2 好友对比玩法

生成：

```text
你：嘉豪型 🤡
朋友：脑内加戏型 🧠

结论：
你负责输出，他负责脑补
```

---

### 5.3 排行榜机制

```text
今日人格TOP：

1️⃣ 嘉豪型（23%）
2️⃣ 脑补型（18%）
```

作用：

* 提高用户参与感
* 增强对比心理

---

### 5.4 隐藏人格

* 条件触发（极端选项组合）
* 或分享后解锁

示例：

* 终极抽象体

---

### 5.5 抽象浓度条

```text
抽象指数：████████░░ 80%
```

增强可视化对比

---

### 5.6 AI文案生成

点击生成：

```text
我是嘉豪型抽象圣体，
我不讲逻辑，我讲感觉。
```

用于社交传播

## 六、技术方案

### 6.1 技术选型

* HTML + CSS + 原生 JavaScript
* 无需框架（MVP阶段）

---

### 6.2 文件结构

```text
/index.html
/style.css
/script.js
/data/questions.js
/data/personalities.js
```

---

### 6.3 核心逻辑

#### 1. 记录用户选择

```js
answers = [option1, option2...]
```

#### 2. 计算维度分数

```js
score = {
  emotion: 0,
  social: 0,
  logic: 0,
  selfView: 0
}
```

#### 3. 转换ABTI编码

```js
emotion > 0 ? X : C
```

#### 4. 匹配人格

* 根据code查表
* 或做距离匹配
